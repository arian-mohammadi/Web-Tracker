# Arian Tracker 🌍

A modern, responsive Web-based GPS Tracking Dashboard built with HTML5, CSS3, Bootstrap 5, and Google Maps JavaScript API. This application provides a clean user interface to monitor devices, manage tracker locations, and analyze user interactions in real-time.

---

## 🚀 Live Demo
> **[⚡ View Web-Tracker Live in Your Browser](https://web-tracker-rho-pied.vercel.app/)**

---

## ✨ Features

- **🔐 User Authentication UI:** Beautiful and clean Login and Registration pages with modern floating labels.
- **🗺️ Interactive Google Map Integration:** Core dashboard featuring an embedded dynamic map with custom markers and info windows.
- **📱 Fully Responsive Layout:** Powered by Bootstrap 5, ensuring a seamless experience across mobile, tablet, and desktop screens.
- **🎨 Custom Styling & RTL Support:** Implemented customized UI cards, custom background overlay effects, and optimized right-to-left (RTL) typography using the IranSans font.
- **⚙️ Device Management Modals:** Built-in popups/modals for adding new tracking devices and configuring settings.

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3
- **Framework:** Bootstrap 5
- **Icons:** FontAwesome v6.1.0
- **API:** Google Maps JavaScript API

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps:

1. **Clone the repository:**
   ```bash
   https://github.com/arian-mohammadi/Web-Tracker.git
   ```

2. **Navigate to the project directory:**
    `cd Web-Tracker`

3. **Open the application:**
Simply open login.html in any web browser to explore the UI.
